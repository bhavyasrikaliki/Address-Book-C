# Address-Book-C

## Description
Address Book Management System implemented using **C programming**.  
This project allows users to store and manage contact details efficiently using file handling.

## Features
- Add new contacts
- Search contacts
- Edit existing contact details
- Delete contacts
- Store data permanently using files

## Technologies Used
- C Programming
- Structures
- File Handling

## How to Run
1. Compile the program:
gcc *.c

2. Run the executable:
./a.out

## Project Use Case
This project is useful for understanding:
- Modular programming in C
- Real-time file operations
- Data management using structures
